# 🤖 InfoThiran AI

### AI-Powered Research Assistant

InfoThiran AI is a smart research assistant designed to help users quickly understand and summarize online content. It simplifies complex information into concise insights, improving productivity and learning efficiency.


## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Spring Boot
* **Database:** MySQL

---

## 📌 How It Works

1. User selects or inputs content
2. The system processes and analyzes the text
3. AI generates a concise summary
4. Output is displayed in a clean, readable format

---

## ⚙️ Installation

```bash
git clone https://github.com/Yuvanesh26012006/research-assistant.git
cd research-assistant
```

### Run Backend (Spring Boot)

```bash
mvn spring-boot:run
```

### Open Frontend

* Open `index.html` in browser
